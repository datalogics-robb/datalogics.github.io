Converts sets of values from a Source ColorSpace to a Destination colorspace using a specified Render Intent.

Parameters
sourceColorProfileThe Color Profile to be used for input values.
destinationColorProfileThe Color Profile to be used for output values.
intentThe Render Intent used during the conversion.
numberOfInputValueSetsThe number of sets of input values to be converted.
inputValueSetsThe input values to be converted which are divided into sets, e.g. < 255 140 0 0 120 0 60 0 > would represent 2 sets of CMYK colors: [255 140 0 0] and [120 0 60 0] and numberOfInputValueSets should be 2.
