Detailed Description

Enumerator used to select type of compression used when images are recompressed to a tighter compression, or following decompression, and downsampling.

For color or grayscale images: Flate, JPEG or JPEG2000 may be selected. For black and white images: Flate, JBIG2, CCITTG3, or CCITTG4 may be selected.
