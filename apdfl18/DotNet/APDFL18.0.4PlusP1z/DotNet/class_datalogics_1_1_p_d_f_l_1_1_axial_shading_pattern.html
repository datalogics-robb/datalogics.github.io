Detailed Description

A color blend that varies along an axis.

An axial shading specifies a color blend that varies along a linear axis between two endpoints, and indefinitely perpendicular to that axis. For more detail on working with axial shading pattern, see section 8.7.4.5.3 of the ISO 32000-1:2008, Document Management-Portable Document Format-Part 1: PDF 1.7, page 185.
